# KikoScript_Keyboard-and-Typeface
The Kiko script keyboard and typeface file

For the MacBook Keyboard ending in ".dmg" all you need to do is download the file and install it.

For Windows, follow these instructions:
1. Download the Universal Keyboard file above
2. Download Keyman on your computer or any device you use here - *right click to open link on a different page* [Keyman Keyboard](https://keyman.com/)
3. Open and install the software
4. Right Click the Universal keyboard file and open with Keyman

**Please note: the Universal keyboard excludes certain additional marks related to punctuation *(when the shift alt/option key is pressed)*. Unfortunately, this issue cannot currently be resolved, as it stems from a limitation within the Keyman software.**
